# R-kasir
POS kasir admin dan petgas kasir
Langkah-Langkah Pemasangan di Google Sheets & Google Apps Script:
Buka halaman Google Sheets dan buat dokumen spreadsheet baru.

Pada menu atas, klik Ekstensi > Apps Script (Editor skrip akan terbuka di tab baru).

Hapus kode default bawaan, lalu buat file baru dengan nama Code.gs dan masukkan isi kode Apps Script di atas.

Buat file baru lagi dengan nama Index.html (pastikan ekstensinya tepat) di editor tersebut, dan masukkan seluruh kode HTML Aplikasi Web R-kasir Profesional yang telah kita buat.

Jalankan fungsi createSheetsSetup() terlebih dahulu dari dalam panel editor Apps Script dengan mengklik Jalankan untuk membuat tabel awal otomatis di Google Sheets Anda.

Klik tombol Terapkan (Deploy) di kanan atas editor, lalu pilih Penerapan Baru.

Pilih jenis penerapan sebagai Aplikasi Web, setel aksesnya menjadi "Siapa saja yang memiliki akun Google" atau "Siapa saja", lalu klik Terapkan.

Salin tautan (URL) Aplikasi Web yang muncul. Buka tautan tersebut untuk mulai menggunakan aplikasi kasir toko R-kasir Anda yang profesional.
